Rendobar is MCP-native -- AI agents can process videos, check job status, and manage their account through the Model Context Protocol without writing HTTP code.

## What is MCP?

The Model Context Protocol (MCP) is a standard for AI agents to interact with external services. Instead of making REST API calls, agents use typed tools with structured inputs and outputs.

## Available MCP tools

Rendobar exposes 5 MCP tools:

### Job management

| Tool | Description |
|------|-------------|
| `submit_job` | Submit a job (today: `raw.ffmpeg`) |
| `get_job` | Get job status, progress, results, and output URL |
| `list_jobs` | List jobs with optional status and type filters |

### Media

| Tool | Description |
|------|-------------|
| `upload_media` | Get upload endpoint for local files (returns URL + instructions) |

### Account

| Tool | Description |
|------|-------------|
| `get_account` | Check balance, plan limits, and active job count |

## Connecting an agent

### Claude Desktop / Claude Code

Add to your MCP config:

```json
{
  "mcpServers": {
    "rendobar": {
      "url": "https://api.rendobar.com/mcp",
      "headers": {
        "Authorization": "Bearer rb_live_YOUR_KEY"
      }
    }
  }
}
```

### Custom agents

Any MCP-compatible agent can connect to Rendobar's MCP endpoint using the Streamable HTTP transport.

## Example: Agent workflow

An AI agent can process videos end-to-end:

<Steps>

<Step title="Check balance">
  Call `get_account` to verify sufficient credits.
</Step>

<Step title="Upload (if needed)">
  Call `upload_media` to get an upload endpoint, then upload the file.
</Step>

<Step title="Submit job">
  Call `submit_job` with type, inputs, and parameters.
</Step>

<Step title="Monitor">
  Call `get_job` to poll for completion.
</Step>

<Step title="Report">
  Present the result (output URL and metadata) to the user.
</Step>

</Steps>

## Authentication

MCP tools authenticate via API key in the `Authorization` header. The same API keys used for the REST API work with MCP. OAuth 2.1 is also supported for MCP clients that handle it natively.
