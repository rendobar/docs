## Credit packs

Credit packs are available for both plans: $10 / $25 / $50 / $100 (or custom, minimum $10).

Pro subscribers receive a 20% bonus on every purchase (e.g., buy $25 of credits, get $30).

Credits never expire. Failed jobs are always refunded.

## API response limits

| Limit | Value |
|-------|-------|
| Default page size | 50 items |
| Maximum page size | 100 items |
| Maximum request body | 100 MB |

## File format support

Rendobar supports all FFmpeg-compatible formats for input. Default output formats:

| Input type | Default output |
|-----------|---------------|
| Video | MP4 (H.264) |
| Audio | MP3 |
| Image | PNG |

Output format can be configured per job type via the `outputFormat` parameter where applicable.
