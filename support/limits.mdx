## Plan limits

Values pulled directly from `PLAN_LIMITS` in the Rendobar API. `-1` means unlimited.

| Limit | Free | Pro ($9/mo) |
|-------|------|-------------|
| Signup credits | $5 (one-time) | — |
| Monthly credits | — | $5/mo included |
| Credit bonus on purchases | — | +20% |
| Concurrent jobs | 1 | 25 |
| API requests / min | 30 | 300 |
| Max input file | 100 MB | 2 GB |
| Max batch size | 10 | 100 |
| Max job timeout | 5 min | 15 min |
| Output retention | 7 days | 30 days |
| Storage quota (total R2) | 1 GB | 50 GB |

## Output URLs and retention

- **Signed download URL** (`outputUrl` on completed jobs) is valid for **1 hour**. Re-fetch the job via `GET /jobs/{id}` to get a fresh URL after it expires.
- **Output retention** is 7 days on Free and 30 days on Pro. After that, Rendobar auto-deletes the file from R2 — the job record remains but `outputUrl` stops resolving.
- **Storage quota** is the total bytes you can hold in R2 across all your outputs. Delete old outputs or upgrade if you hit the cap.
- **Output file size** is bounded by the input file size cap and the storage quota — there's no separate per-output limit.

## Credit packs

Credit packs are available for both plans: $10 / $25 / $50 / $100 (or custom, minimum $10).

Pro subscribers receive a 20% bonus on every purchase (e.g., buy $25 of credits, get $30).

Credits never expire. Failed jobs are always refunded.

## File format support

Rendobar supports all FFmpeg-compatible formats for input. Default output formats:

| Input type | Default output |
|-----------|---------------|
| Video | MP4 (H.264) |
| Audio | MP3 |
| Image | PNG |

For `raw.ffmpeg`, Rendobar infers the output format from the trailing filename in your command (e.g., `output.mp4` → MP4). Override it with the `outputFormat` param if needed.
