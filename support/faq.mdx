<Accordion title="How much does Rendobar cost?">
  Rendobar uses credit-based billing. The free tier includes a one-time $5 signup grant with no credit card required. The Pro plan is $9/month and includes $5/month in credits plus a 20% bonus on every credit purchase. Credit packs ($10–$100) are available on both plans. Credits never expire.
</Accordion>

<Accordion title="What can I do with Rendobar today?">
  The live API is `raw.ffmpeg` — run any whitelisted FFmpeg command against URLs you provide. That one job type covers transcoding, trimming, filtering, overlays, audio extraction, frame extraction, and anything else FFmpeg can do from the command line. More dedicated job types are on the roadmap.
</Accordion>

<Accordion title="What video formats are supported?">
  Rendobar supports every FFmpeg-compatible container and codec. Outputs default to the extension in your `-i` target filename (e.g., `output.mp4` → MP4).
</Accordion>

<Accordion title="Can I use Rendobar from AI agents?">
  Yes. Rendobar is MCP-native. Connect any MCP-compatible AI agent to `api.rendobar.com/mcp` with your API key. The MCP tools cover job submission, status polling, file upload, and account management. MCP is available on all plans, including Free.
</Accordion>

<Accordion title="How long do jobs take?">
  Most jobs complete in seconds to minutes depending on input size and the command. Maximum timeout varies by plan: 5 minutes on Free, 15 minutes on Pro.
</Accordion>

<Accordion title="Is there a rate limit?">
  Yes. The Free plan allows 30 API requests per minute. The Pro plan allows 300 requests per minute. Rate limits apply to both REST API and MCP tool calls.
</Accordion>

<Accordion title="What happens if a job fails?">
  No credits are charged for failed jobs. You can retry failed jobs via the API or MCP.
</Accordion>

<Accordion title="Can I run custom FFmpeg commands?">
  Yes — that is the core `raw.ffmpeg` job type. Your command is validated against a flag whitelist and security layers before execution in an isolated container. Raw FFmpeg is available on all plans, including Free.
</Accordion>

<Accordion title="How do I get real-time job updates?">
  The dashboard receives real-time updates via WebSocket (OrgHub). For programmatic use, configure a webhook endpoint on your organization for push-based notifications, or poll `GET /jobs/{id}`.
</Accordion>
