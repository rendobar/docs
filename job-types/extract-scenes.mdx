Analyze a video and detect scene change boundaries. Returns timestamps and confidence scores for each detected scene transition.

<Info>
**Timeout:** 300s | **Provider:** triggerdev | **Accepts:** video
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "extract.scenes",
  "inputs": {
    "source": "https://cdn.rendobar.com/assets/examples/sample.mp4"
  },
  "params": {
    "threshold": 0.3
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    type: "extract.scenes",
    inputs: {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
    params: {"threshold":0.3},
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/jobs",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "type": "extract.scenes",
        "inputs": {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
        "params": {"threshold":0.3},
    },
)
```

</CodeGroup>

## Parameters

<ParamField body="threshold" type="number" default="0.4">
  Confidence threshold (0-1)
</ParamField>

<ParamField body="minDuration" type="number" default="0.5">
  Minimum scene duration in seconds (ignore shorter scenes)
</ParamField>
