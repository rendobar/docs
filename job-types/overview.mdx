Rendobar organizes media processing into job types. Each job type has a specific Zod schema for input validation, a cost model, and execution requirements. Submit any job type via the REST API or MCP tools.

## Live Now

These job types are available in production:

| Job Type | Description | Cost |
|----------|-------------|------|
| [raw.ffmpeg](/job-types/raw-ffmpeg) | Execute arbitrary FFmpeg commands with safety guardrails | ~$0.05/min compute |
| [extract.metadata](/job-types/extract-metadata) | Extract comprehensive metadata from media files (also available via MCP `probe_media`) | $0.02 flat |

These job types are defined and will be available shortly:

| Job Type | Description | Estimated Cost |
|----------|-------------|----------------|
| watermark.apply | Add a single image watermark to video or image | ~$0.50/min (video), $0.20 flat (image) |
| watermark.composite | Overlay multiple watermarks in a single pass | ~$0.50/min (video), $0.20 flat (image) |
| watermark.remove | Remove a watermark using AI detection or manual regions | ~$0.50/min (video), $0.20 flat (image) |
| watermark.check | Analyze frames for watermark presence using AI | $0.10 flat |
| watermark.embed | Embed an invisible watermark using VideoSeal | $0.03 flat |
| watermark.detect | Detect and decode an invisible watermark | $0.03 flat |

### QR Code Generation

| Job Type | Description | Estimated Cost |
|----------|-------------|----------------|
| qr.generate | Generate styled QR codes with 9 content types and custom styling | $0.02 flat |

**QR content types:** `url`, `text`, `phone`, `sms`, `email`, `vcard`, `wifi`, `geo`, `whatsapp`

**QR style options:** size (64-2048px), margin, error correction (L/M/Q/H), foreground/background color, dot style (square/rounded/dots/extra-rounded), format (png/svg), optional logo with auto-upgraded error correction.
