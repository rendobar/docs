Rendobar supports 23 job types organized into categories. Each job type has a specific Zod schema for input validation, a cost model, and execution requirements. Most job types accept multiple media formats -- the system automatically detects whether your input is video, image, or audio and adjusts processing, output format, and billing accordingly.

## Watermarking

| Job Type | Description | Cost |
|----------|-------------|------|
| [watermark.add](/job-types/watermark-apply) | Add a single image watermark to a video or image | ~$0.50/min (video), $0.20 flat (image) |
| [watermark.composite](/job-types/watermark-apply) | Overlay multiple watermarks (images and text) in a single pass | ~$0.50/min (video), $0.20 flat (image) |
| [watermark.remove](/job-types/watermark-remove) | Remove a watermark using AI detection or manual region specification | ~$0.50/min (video), $0.20 flat (image) |
| [watermark.check](/job-types/watermark-check) | Analyze frames for watermark presence using AI classification | $0.10 flat |
| [watermark.embed](/job-types/watermark-embed) | Embed an invisible watermark using VideoSeal | $0.03 flat |
| [watermark.detect](/job-types/watermark-detect) | Detect and decode an invisible watermark | $0.03 flat |

## Transcoding

| Job Type | Description | Cost |
|----------|-------------|------|
| [transcode](/job-types/transcode) | Convert between video or audio formats, adjust codecs, bitrate, resolution | ~$0.50/min (video), $0.20 flat (audio) |

## Compliance (AI-powered)

| Job Type | Description | Cost |
|----------|-------------|------|
| [compliance.moderate](/job-types/compliance-moderate) | Content moderation for video or image using AI frame analysis | ~$0.50/min (video), $0.10 flat (image) |
| [compliance.gdpr.detect](/job-types/compliance-gdpr-detect) | Detect faces and license plates for GDPR compliance | ~$0.50/min (video), $0.10 flat (image) |
| [compliance.gdpr.redact](/job-types/compliance-gdpr-redact) | Automatically blur or pixelate detected faces and plates | ~$2.00/min (video), $0.50 flat (image) |

## QR Code Generation

| Job Type | Description | Cost |
|----------|-------------|------|
| [qr.generate](/job-types/qr-generate) | Generate styled QR codes with 9 content types and custom styling | $0.10 flat |

**Content types:** `url`, `text`, `phone`, `sms`, `email`, `vcard`, `wifi`, `geo`, `whatsapp`

**Style options:** size (100--2000px), margin, error correction (L/M/Q/H), foreground/background color, dot style (square/rounded/dots/extra-rounded), format (png/svg), optional logo with auto-upgraded error correction.
