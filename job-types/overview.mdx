Rendobar organises media processing as job types. Each type has a Zod schema for input validation, a cost model, and execution requirements. Submit any job type via `POST /jobs` or through an MCP tool.

## Live

Production-ready. Use these on all plans.

| Job type | Summary | Cost |
|---|---|---|
| [raw.ffmpeg](/job-types/raw-ffmpeg) | Execute a custom FFmpeg command with flag whitelisting and sandboxing | ~$0.05/min compute |

## Preview

Functional but still stabilising. The surface may change before GA. Available on request -- email `support@rendobar.com` if you need early access or hit issues.

| Job type | Summary | Estimated cost |
|---|---|---|
| [watermark.apply](/job-types/watermark-apply) | Overlay up to 10 image/text watermarks on a video or image | ~$0.50/min (video), $0.20 flat (image) |
| [watermark.check](/job-types/watermark-check) | AI scan of image frames for watermark presence | $0.10 flat |
| [watermark.detect](/job-types/watermark-detect) | Extract or verify an invisible watermark via VideoSeal | $0.03 flat per image |
| [watermark.embed](/job-types/watermark-embed) | Embed an invisible tracking payload via VideoSeal | $0.03 flat per image |
| [watermark.remove](/job-types/watermark-remove) | Remove a watermark using AI detection or manual regions | ~$0.50/min (video), $0.20 flat (image) |
| [qr.generate](/job-types/qr-generate) | Generate a styled QR code (9 content types, logo embed) | $0.02 flat |

## Coming soon

Defined in the API but not yet routable -- submitting returns `INVALID_JOB_TYPE`. These pages describe the upcoming shape so you can build against it.

### Transcoding

| Job type | Summary |
|---|---|
| [transcode](/job-types/transcode) | Convert codec, container, resolution, frame rate |
| [thumbnail](/job-types/thumbnail) | Extract a single frame from a video, or resize an image |

### Captions

| Job type | Summary |
|---|---|
| [caption.extract](/job-types/caption-extract) | Transcribe speech to SRT, VTT, or JSON (Deepgram) |
| [caption.burn](/job-types/caption-burn) | Permanently render subtitles onto a video |

### Compliance

| Job type | Summary |
|---|---|
| [compliance.moderate](/job-types/compliance-moderate) | Scan for violence, nudity, hate speech, self-harm, drugs, weapons, graphic content |

| [compliance.gdpr.redact](/job-types/compliance-gdpr-redact) | Redact detected PII with blur, pixelate, or blackbox |

| Job type | Summary |
