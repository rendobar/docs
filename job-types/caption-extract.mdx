Transcribe speech from a video or audio file using AI (Deepgram) and output captions in SRT, VTT, or JSON format.

<Info>
**Timeout:** 300s | **Provider:** triggerdev | **Accepts:** video, audio
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "caption.extract",
  "inputs": {
    "source": "https://cdn.rendobar.com/assets/examples/sample.mp4"
  },
  "params": {
    "language": "en",
    "format": "srt"
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    type: "caption.extract",
    inputs: {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
    params: {"language":"en","format":"srt"},
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/jobs",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "type": "caption.extract",
        "inputs": {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
        "params": {"language":"en","format":"srt"},
    },
)
```

</CodeGroup>

## Parameters

<ParamField body="language" type="string" default="en">
  Language code for transcription (BCP-47, e.g. "en", "es", "fr")
</ParamField>

<ParamField body="format" type="srt | vtt | json" default="srt">
  Output caption format
</ParamField>
