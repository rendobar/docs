Render a video from an existing VEED project by project ID. Supports MP4, WebM, and MOV output formats.

<Info>
**Timeout:** 600s | **Provider:** triggerdev | **Accepts:** none (VEED project by ID)
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "render.veed",
  "inputs": {},
  "params": {
    "projectId": "veed-project-abc123",
    "outputFormat": "mp4",
    "quality": "standard"
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    type: "render.veed",
    inputs: {},
    params: {"projectId":"veed-project-abc123","outputFormat":"mp4","quality":"standard"},
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
```

</CodeGroup>

## Parameters

<ParamField body="projectId" type="string" required>
  VEED project ID
</ParamField>

<ParamField body="outputFormat" type="mp4 | webm | mov" default="mp4">
  Output video format
</ParamField>

<ParamField body="quality" type="draft | standard | high" default="standard">
  Quality preset
</ParamField>


## Response

```json
{
  "data": {
    "id": "job_abc123",
    "status": "dispatched"
  }
}
```
