Generate a photorealistic product mockup by replacing smart object layers in a PSD template with your design images. Uses Photopea for full PSD rendering fidelity — all layer effects, shadows, textures, and perspective transforms are preserved. Supports placement controls (position, scale, rotation) and multiple layer replacements per template.

<Info>
**Timeout:** 120s | **Provider:** triggerdev | **Accepts:** PSD template (via `inputs.template`)
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "mockup.render",
  "inputs": {
    "template": "tmpl_example123"
  },
  "params": {
    "layers": [
      {
        "name": "Screen",
        "image": "https://cdn.rendobar.com/assets/brand/logo-full.png",
        "placement": {
          "x": 0,
          "y": 0,
          "scale": 1,
          "rotation": 0
        }
      }
    ]
  }
}'
```
