Automatically detect and redact faces, license plates, and text from a video or image. Supports blur, pixelate, and blackbox redaction methods.

<Info>
**Timeout:** 3600s | **Provider:** triggerdev | **Accepts:** video, image
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/gdpr/redact \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "inputs": {
    "source": "https://cdn.rendobar.com/assets/examples/sample.mp4"
  },
  "params": {
    "redactTypes": [
      "faces",
      "license_plates"
    ],
    "method": "blur"
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/gdpr/redact", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    inputs: {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
    params: {"redactTypes":["faces","license_plates"],"method":"blur"},
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/gdpr/redact",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "inputs": {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
        "params": {"redactTypes":["faces","license_plates"],"method":"blur"},
    },
)
```

</CodeGroup>

## Parameters

<ParamField body="redactTypes" type=""faces" | "license_plates" | "text"[]" default="["faces","license_plates"]">
  Types of PII to redact
</ParamField>

<ParamField body="method" type=""blur" | "pixelate" | "blackbox"" default=""blur"">
  Redaction method
</ParamField>

<ParamField body="blurStrength" type="integer" default="30">
  Blur strength when method is blur (1-100)
</ParamField>

<ParamField body="threshold" type="number" default="0.5">
  Confidence threshold (0-1)
</ParamField>

<ParamField body="padding" type="integer" default="10">
  Padding around detected regions in pixels
</ParamField>
