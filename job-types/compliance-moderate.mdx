Scan a video or image for policy-violating content (violence, nudity, hate speech, etc.) using AI frame analysis. Returns per-frame confidence scores.

<Info>
**Timeout:** 300s | **Provider:** cf-ai | **Accepts:** video, image
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "compliance.moderate",
  "inputs": {
    "source": "https://cdn.rendobar.com/assets/examples/sample.mp4"
  },
  "params": {
    "threshold": 0.5,
    "sampleInterval": 2
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {

  body: JSON.stringify({

});
const { data } = await response.json();
```


## Parameters

<ParamField body="categories" type="violence | nudity | hate | self-harm | drugs | weapons | graphic[]" default="["violence","nudity","hate","self-harm"]">
  Content categories to check for
</ParamField>

<ParamField body="threshold" type="number" default="0.5">
  Confidence threshold (0-1)
</ParamField>

<ParamField body="sampleInterval" type="number" default="2">
  Seconds between sampled frames
</ParamField>
