Execute a custom FFmpeg command with whitelisted flags. Input files are URLs in `-i` positions, downloaded and resolved securely. Available on all plans.

<Info>
**Timeout:** 900s | **Provider:** triggerdev | **Accepts:** video, audio, image
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "raw.ffmpeg",
  "inputs": {},
  "params": {
    "command": "ffmpeg -i https://cdn.rendobar.com/assets/examples/sample.mp4 -vf scale=1280:720 -c:v libx264 -crf 23 -preset fast output.mp4"
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    type: "raw.ffmpeg",
    inputs: {},
    params: {
      command: "ffmpeg -i https://cdn.rendobar.com/assets/examples/sample.mp4 -vf scale=1280:720 -c:v libx264 -crf 23 -preset fast output.mp4",
    },
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/jobs",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "type": "raw.ffmpeg",
        "inputs": {},
        "params": {
            "command": "ffmpeg -i https://cdn.rendobar.com/assets/examples/sample.mp4 -vf scale=1280:720 -c:v libx264 -crf 23 -preset fast output.mp4",
        },
    },
)
```

</CodeGroup>

## Parameters

<ParamField body="command" type="string" required>
  Real FFmpeg command starting with `ffmpeg`. Input URLs go in `-i` positions.
</ParamField>

<ParamField body="outputFormat" type="mp4 | mkv | webm | mov | avi | ts | gif | png | jpg | mp3 | wav | flac | ogg | aac | opus | m4a | srt | vtt" default="mp4">
  Output container/format. Inferred from trailing filename if not set.
</ParamField>

<ParamField body="timeout" type="integer" default="120">
  Maximum execution time in seconds (1-900)
</ParamField>


## Response

```json
{
  "data": {
    "id": "job_abc123",
    "status": "dispatched"
  }
}
```
