Detect personally identifiable information in a video or image, including faces and license plates. Returns timestamped detection coordinates.

<Info>
**Timeout:** 300s | **Provider:** triggerdev | **Accepts:** video, image
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "compliance.gdpr.detect",
  "inputs": {
    "source": "https://cdn.rendobar.com/assets/examples/sample.mp4"
  },
  "params": {}
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    type: "compliance.gdpr.detect",
    inputs: {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
    params: {},
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/jobs",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "type": "compliance.gdpr.detect",
        "inputs": {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
        "params": {},
    },
)
```

</CodeGroup>

## Parameters

<ParamField body="detectTypes" type=""faces" | "license_plates" | "text"[]" default="["faces","license_plates"]">
  Types of PII to detect
</ParamField>

<ParamField body="threshold" type="number" default="0.5">
  Confidence threshold (0-1)
</ParamField>

<ParamField body="sampleInterval" type="number" default="0.5">
  Seconds between sampled frames
</ParamField>
