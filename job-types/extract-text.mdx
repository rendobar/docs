Extract visible text from video frames or a single image using OCR. Returns timestamped text detections with bounding boxes.

<Info>
**Timeout:** 300s | **Provider:** triggerdev | **Accepts:** video, image
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "type": "extract.text",
  "inputs": {
    "source": "https://cdn.rendobar.com/assets/examples/sample.mp4"
  },
  "params": {
    "sampleInterval": 2
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    type: "extract.text",
    inputs: {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
    params: {"sampleInterval":2},
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/jobs",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "type": "extract.text",
        "inputs": {"source":"https://cdn.rendobar.com/assets/examples/sample.mp4"},
        "params": {"sampleInterval":2},
    },
)
```

</CodeGroup>

## Parameters

<ParamField body="sampleInterval" type="number" default="1">
  Seconds between sampled frames
</ParamField>

<ParamField body="language" type="string" default=""en"">
  OCR language hint (BCP-47 code)
</ParamField>

```json
{
```
