Analyze image frames to detect the presence of watermarks using AI classification. Returns confidence scores, bounding boxes, and a summary with dominant watermark region.

<Info>
**Timeout:** 10s | **Provider:** cf-ai | **Accepts:** video
</Info>

## Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/check \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
  "inputs": {
    "frames": [
      "https://cdn.rendobar.com/assets/examples/sample-frame.jpg"
    ]
  },
  "params": {
    "threshold": 0.5
  }
}'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/check", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    inputs: {"frames":["https://cdn.rendobar.com/assets/examples/sample-frame.jpg"]},
    params: {"threshold":0.5},
  }),
});
const { data } = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/check",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "inputs": {"frames":["https://cdn.rendobar.com/assets/examples/sample-frame.jpg"]},
        "params": {"threshold":0.5},
    },
)
```

</CodeGroup>
