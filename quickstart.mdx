Submit an FFmpeg command to Rendobar, poll the job, download the result. Four steps, under five minutes, no infrastructure.

## 1. Get your API key

Sign up at [app.rendobar.com](https://app.rendobar.com) and create an API key from the dashboard. Keys are prefixed with `rb_`. You get $5 in free credits on signup.

## 2. Submit a job

Submit a `raw.ffmpeg` job with a real FFmpeg command:

<CodeGroup>

```bash cURL
curl -X POST https://api.rendobar.com/jobs \
  -H "Authorization: Bearer rb_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "raw.ffmpeg",
    "inputs": {},
    "params": {
      "command": "ffmpeg -i https://example.com/video.mp4 -vf scale=1280:720 -c:v libx264 -preset fast -crf 23 output.mp4"
    }
  }'
```

```javascript JavaScript
const response = await fetch("https://api.rendobar.com/jobs", {
  method: "POST",
  headers: {
    "Authorization": "Bearer rb_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    type: "raw.ffmpeg",
    inputs: {},
    params: {
      command: "ffmpeg -i https://example.com/video.mp4 -vf scale=1280:720 -c:v libx264 -preset fast -crf 23 output.mp4",
    },
  }),
});
const { data } = await response.json();
console.log(data.id); // job_abc123
```

```python Python
import requests

response = requests.post(
    "https://api.rendobar.com/jobs",
    headers={"Authorization": "Bearer rb_your_key"},
    json={
        "type": "raw.ffmpeg",
        "inputs": {},
        "params": {
            "command": "ffmpeg -i https://example.com/video.mp4 -vf scale=1280:720 -c:v libx264 -preset fast -crf 23 output.mp4",
        },
    },
)
data = response.json()["data"]
print(data["id"])  # job_abc123
```

</CodeGroup>

Put input URLs directly in `-i` flags. The output format is inferred from the trailing filename (`output.mp4` → MP4).

The response includes a job ID and status:

```json
{
  "data": {
    "id": "job_abc123",
    "status": "dispatched"
  }
}
```

## 3. Check job status

Poll the job endpoint until the status is `complete`:

```bash
curl https://api.rendobar.com/jobs/job_abc123 \
  -H "Authorization: Bearer rb_your_key"
```

## 4. Download the result

Once complete, the job response includes a download URL for the processed file.
